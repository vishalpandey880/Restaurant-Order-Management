# Restaurant Management System (C++)

A simple console-based Restaurant Management System written in C++. This program allows users to view a menu, place orders, and generate a final bill.

## Features

- **Show Menu:** Displays the available food items along with their IDs and prices.
- **Order Food:** Allows the user to select items by ID and specify the quantity.
- **View Bill:** Generates a detailed receipt showing individual item totals and the final total bill.
- **Exit:** Closes the application.
